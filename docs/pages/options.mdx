# Options

## Custom the Feedback Button

Wrap whatever button or item you want to trigger the widget:

```tsx
<Feedback>
  <button>Send Feedback</button>
</Feedback>
```

## Submitters email

Pass the email of the currently authenticated user to the `user` prop:

```tsx
// ...
import { Feedback } from '@okie/feedback';

const App = () => {
    
  return (
    <Layout>
      <Feedback user={currentUser.email} />
    </Layout>
  );
}
```

## Build your own form

You can build your own form and submit the feedback to our API endpoint. 
Send a POST request to [https://feedback.okie.one/api/feedback/create](https://feedback.okie.one/api/feedback/create):

```js
fetch('https://feedback.okie.one/api/feedback', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    projectId: "...",
    url: 'https://your-url',
    email: 'your@email.com',
    name: 'Your Name',
    message: 'Feedback Message',
    screenshot: 'url to screenshot image, or base64'
    metadata: '{}'
  })
})
```
